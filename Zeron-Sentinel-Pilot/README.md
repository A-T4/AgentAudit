# The Zeron Sentinel Pilot

**Infrastructure-Level Admission Control & Deep Packet Inspection for the Zeron ADK**

## The Threat Landscape

Agent frameworks like the Zeron Agent Development Kit (ZAK) currently rely on verb-based, payload-blind Role-Based Access Control (RBAC). The policy engine evaluates the *tool name* (e.g., `execute_shell`) but fails to inspect the *tool arguments* generated by the LLM. 

This creates a critical evasion vector: an attacker can bypass standard runtime guardrails by passing obfuscated payloads (e.g., Base64-encoded reverse shells) through an authorized tool. 

## The Sentinel Architecture

The **AgentAudit Sentinel** is an in-line admission controller that hooks directly into the ZAK `AgentExecutor`. It enforces Zero Trust Architecture by applying Deep Packet Inspection (DPI) to the LLM's data stream *before* execution is permitted.

### Core Enforcement Mechanisms

1.  **Recursive Decoding Pipeline:** Automatically strips Base64, Hex, and URL encodings from LLM arguments to expose the raw payload.
2.  **Cryptographic Secret Detection:** Calculates the Normalized Shannon Entropy of the decoded arguments to identify and drop obfuscated exfiltration strings:
    $$H_{rel} = \frac{-\sum_{i=1}^{n} p_i \log_2(p_i)}{\log_2(N)}$$
    *(Payloads scoring $H_{rel} > 0.85$ are instantly blocked).*
3.  **Data Principal Protection (DPDP Act):** Parallelized RegEx engines scan for regional PII (Aadhaar, PAN) to enforce strict localized data residency compliance at the edge.
